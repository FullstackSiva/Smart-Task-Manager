# Smart Task Manager

A premium productivity dashboard built using Angular.

## Live Demo

[View Live Project](https://6a0f6e514d9c0322f8f74d4f--unrivaled-semolina-915d38.netlify.app/)

## Features

- Add / Edit / Delete Tasks
- Task Archive System
- Task Analytics Charts
- Drag & Drop Kanban Board
- Due Date Reminder System
- Dark Mode
- Search & Filters
- Responsive Design
- Local Storage Persistence
- Authentication UI
- Premium Dashboard UI

## Tech Stack

- Angular
- TypeScript
- RxJS
- Chart.js
- Angular CDK Drag & Drop
- HTML5
- CSS3
